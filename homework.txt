Дз сервис поиска работы

1. Создать spring-boot проект. Выбрать зависимости и плагины на свое усмотрение.
2. Создать сервис размещения вакансий.
 У сервиса может быть два профиля: japan(валюта - йена) и china (валюта - Юань). Сервис умеет:
 2.1. Создавать компанию (наименование компании)
 2.2. Создавать вакансию внутри компании
 (наименование компании, наименование должности, описание вакансии, заработная плата, валюта из ресурсов)
 2.3. Отдавать список вакансий по компании.
3. Создать сервис размещения резюме. Сервис умеет:
 3.1 Создавать пользователя(имя, email)
 3.2 Создавать резюме у пользователя (зарплата на которую претендует пользователь, описание того что умеет).
 3.3 Отдавать список резюме.
4. Создать сервис поиска работы. Он будет содержать в себе сервисы из 2 и 3 пунктов и вызывать методы этих сервисов. Т.е. будет 6 методов внутри этого сервиса.
5. Написать тесты (с поднятием тестового контекста) на все методы сервиса поиска работы.
